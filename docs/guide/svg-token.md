# SVG Canarytoken

## What is an SVG Canarytoken

This Canarytoken is an SVG image which contains a 1px square transparent GIF as an external resource. When the SVG is rendered (e.g. by viewing it in a browser), you receive an alert.

## Creating the Canarytoken

Create a Canarytoken by choosing "SVG Image" from the Canarytokens list.

Enter your email and leave a reasonable comment to remind yourself where you will deploy the Canarytoken.

Click "Create Canarytoken", and either copy or download the SVG.

## How to use this Canarytoken

This Canarytoken can be used anywhere SVGs are a valid image format. Embed the SVG directly into a web page, or download it to leave it in your Dropbox, local Desktop/Documents folder, or host it somewhere else.
